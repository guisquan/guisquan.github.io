在Ubuntu上安装Docker

准备

操作系统要求

需要一个以下版本的64位系统：

  • Ubuntu Focal 20.04 (LTS)
  • Ubuntu Bionic 18.04 (LTS)
  • Ubuntu Xenial 16.04 (LTS)

卸载旧版本

  1. 在 https://download.docker.com/linux/ubuntu/dists/, 选择你的Ubuntu版本, 在pool/stable/ 并选择amd64，armhf，ppc64el，或者s390x。
